# 🎮 Context-Aware Gaming Toxicity Detector

An NLP-based desktop application designed to detect toxic behavior in online multiplayer gaming environments (specifically *Mobile Legends: Bang Bang* and *Dota 2*). This project focuses on analyzing sequential chat context and utilizes a localized dataset featuring Malay and Manglish (Malaysian English) gamer slang.

## ✨ Key Features

* **Context-Aware Classification:** Merges previous chat history (T-1) with the current message (T) to understand multi-turn intent and avoid misclassifying normal banter.
* **Low-Resource Language Support:** Trained specifically to recognize Malaysian gaming slang, abbreviations, and localized profanity.
* **Dual Model Pipeline:** Evaluates both a baseline Multinomial Naive Bayes model and an optimized Logistic Regression classifier.
* **Real-Time GUI:** Includes a built-in, dark-themed interactive interface built with `tkinter` for live message testing.
* **Automated Visualizations:** Generates and saves a Confusion Matrix and Model Comparison chart upon execution.

## 📁 Repository Structure

* `toxicity_detection.py`: The core Python script containing the data preprocessing, model training pipeline, evaluation metrics, and GUI application.
* `GUI_toxicity_detection.py`: same as the Python script but adding GUI to make it look more interactive.
* `Toxicity Dataset.csv`: The localized dataset containing 1,000 sequential chat samples (Binary Labeling: 0 = Safe, 1 = Toxic).
* `confusion_matrix.png` & `model_comparison_chart.png`: Visual outputs generated by the script during evaluation.
